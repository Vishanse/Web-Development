How to Add a Squarespace Domain to GitHub Pages

Step 1: Log in to your Squarespace account and go to Settings → Domains. Select the domain you want to connect to GitHub Pages.

Step 2: Find the DNS settings for your domain.

Step 3: Add the following DNS records to point your domain to GitHub Pages:

  • A Records: Point to GitHub Pages IP addresses:
    • 185.199.108.153
    • 185.199.109.153
    • 185.199.110.153
    • 185.199.111.153
  • CNAME Record: If using a subdomain (like www.yourdomain.com), point it to yourusername.github.io.

Step 4: In your GitHub repository, create a file named CNAME in the root (same as index.html) and add your full domain (e.g., www.yourdomain.com) inside it. Save and push the file.

Step 5: Wait for DNS propagation (usually a few minutes to a few hours). Then open your domain in a browser — it should point to your GitHub Pages site.

Screenshot placeholder: Squarespace DNS settings and GitHub Pages CNAME file

Tip: Make sure there are no conflicting DNS records (like old forwarding rules) that could prevent your domain from pointing to GitHub Pages.
