Steps to connect Squarespace Domain to GitHub Pages

Connect your squarespace purchased domain to GitHub hosting repository by changing dns settings in squarespace dashboard.

Watch the Tutorial

STEPS

1: Go to squarespace.com and click Domains.

2: Select your purchased domain ( eg: sitewebdevelopment.com)

3: Click DNS >

  • Delete squarespace default record.
  • Keep these records,Squarespace Domain Connect record and Email security record.

4:Click Add Record button.

5: Insert

  • Host-@, Type-A, TTL- 4hrs, Data- 185.199.108.153
  • Host-@, Type-A, TTL- 4hrs, Data- 185.199.109.153
  • Host-@, Type-A, TTL- 4hrs, Data- 185.199.110.153
  • Host-@, Type-A, TTL- 4hrs, Data- 185.199.111.153
  • Host-www, Type-CNAME, TTL- 4hrs, Data- username.github.io

6: Go to your Github Repository and create a flie called CNAME and type www.sitewebdevelopment.com in it and commit changes.

7: Next click settings > pages. In custom domain section insert www.sitewebdevelopment.com and save it. After 24 hours tick the Enforce https.
