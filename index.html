Step 1: Choose Your Code Editor (VS Code Optional)

You can use VS Code for an easy and flexible coding experience, but any text editor works. Type your website address in your project files so visitors can reach your website once it’s live.

Watch the Tutorial
Photo by Marvin meyer on Unsplash

Step 2: Upload Your Website to GitHub

Open your project in VS Code or your preferred editor. Create a new repository on GitHub, then upload your website files. Once uploaded, enable GitHub Pages in repository settings to make your website live.

Watch the Tutorial
Photo by Mad Fish Digital on Unsplash

Step 3: Connect Your Domain to GitHub

After the website hosted in GitHub, Squarespace domain and GitHub repository can combine together. So, your website goes live and visitors around the world can search your website in the internet.
