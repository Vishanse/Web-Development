Things need to be know when designing a website(For Performance)

1: This image file is larger than it needs to be (1658x392) for its displayed dimensions (543x128). Use responsive images to reduce the image download size.

2: Reduce unused rules from stylesheets

3: Set an explicit width and height on image elements

4: Ensure text is consistently visible

5: Displayed dimensions (310x73). Use responsive images to reduce the image download size.

6: Displayed dimensions for logos (80x80). Use responsive images.

7: Low-contrast text is difficult or impossible for many users to read

8: Link text (and alternate text for images, when used as links) that is discernible, unique, and focusable improves the navigation experience for screen reader users.

Use " @media (max-width: 768px) {} " for your css file for mobile and deskstop reponse.
