How to Add a Sitemap to Your GitHub Pages Site

Discover how to create and add a sitemap to your GitHub Pages website. A sitemap helps search engines index your pages more efficiently, improving your website’s SEO and ensuring your content is discoverable by users.

Watch the Tutorial

Step 1: Create a sitemap.xml file in your repository's root folder (where index.html is).

Step 2: Add your website URLs inside it using this format:

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://yourusername.github.io/</loc>
    <lastmod>2025-11-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://yourusername.github.io/about.html</loc>
    <changefreq>monthly</changefreq>
  </url>
</urlset>

Step 3: Save and push the file to your GitHub repository.

Step 4: To verify it’s working, open: https://yourusername.github.io/sitemap.xml

Tip: You can submit this sitemap to Google Search Console for better indexing.
