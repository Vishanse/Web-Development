Lets add a favicon to your site

From this tutorial you are able to understand how to add a favicon to your website in few minutes.

Watch the Tutorial

STEPS

1: Visit Microsoft Designer(Design your favicon )

2: Save your Design in the file manager in your computer as png.

3: Go to squoosh.app/ type squoosh in the web browser.

4: Drag and drop your design to squoosh.app from file manager.

5: In the right side panel ,

  • Click resize
  • Set width to 32 pixels and height to 32 pixels (or 48*48 pixels).
  • Adjust quality to 75-100.
  • Click download button( Image will download as webp).

6: Now go to Github image folder.

7: Click Add file > Upload files.

8: Drag and drop webp image to the folder.

9: In commit changes bar type initial upload.

10: Select commit directly to the main branch.

11: Click commit changes button.

12: Now add the image to every .html file
